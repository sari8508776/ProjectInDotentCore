It's designed not just to serve data, but to provide deep visibility into system performance and user activity.From custom-built logging middleware that tracks every request's duration to secure role-based access control, this API serves as a solid foundation for enterprise-level applications.✨ Key Features🔐 Secure Authentication – Full JWT Bearer implementation with custom policies (e.g., Admin-only access).📊 Advanced Request Logging – Custom Middleware that captures controller actions, user identities, and execution time, processed efficiently via background workers.⚡ Real-time Connectivity – Integrated with SignalR (ActivityHub) for live updates.📁 Static File Hosting – Ready to serve frontend assets (Login, Dashboard) directly from the server.📝 Structured Logging – Powered by Serilog with daily rolling file sinks and console output.🛠 Interactive API Docs – Fully configured Swagger UI with JWT authorization support for easy testing.🛠 Tech StackCategoryTechnologyFrameworkASP.NET Core 8.0LanguageC#LoggingSerilog, Custom MiddlewareSecurityJWT Bearer AuthenticationDocumentationSwashbuckle (Swagger)Real-timeSignalR🏗 Architecture InsightsThe project follows a clean and scalable architecture:Middleware Layer: Intercepts requests to monitor performance and log user actions.Background Workers: Handles asynchronous logging tasks to ensure zero impact on API response times.Dependency Injection: Utilizes a clean AddProjectServices() extension pattern to keep Program.cs organized.🚀 Getting StartedPrerequisites.NET 8 SDKVisual Studio 2022 or VS CodeInstallationClone the repository:Bashgit clone https://github.com/sari8508776/ProjectInDotentCore.git
